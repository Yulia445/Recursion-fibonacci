# Recursion-fibonacci

## Опис
1) Fibonacci без рекурсії
Спочатку я створюю масив sum з двома першими числами Фібоначчі — 0 і 1.
Потім у циклі додаю нові числа, кожне з яких дорівнює сумі двох попередніх. Якщо масив стає довший, ніж потрібно, я видаляю останній елемент, щоб лишилась правильна кількість чисел.

2) Fibonacci з рекурсією
Тут я створила функцію, яка сама себе викликає.
Вона кожного разу записує наступне число Фібоначчі і зменшує лічильник guantity, доки він не стане 0. У результаті виходить масив з потрібною кількістю чисел.
В кінці я викликаю обидві функції з різними значеннями: 3, 5 і 7, щоб перевірити результат.
